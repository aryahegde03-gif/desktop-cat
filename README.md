# 🐈 Desktop Cat

A tiny pixel-art desktop pet built with Electron.

This project is part of my **Daily Build Challenge**, where I build one small project every day instead of doomscrolling.

## ✨ Current Features

- Transparent desktop window
- Pixel-art cat
- Always-on-top desktop companion
- Frameless Electron window
- Foundation for future animations and interactions

## 🚧 Planned Features

- Random walking around the screen
- Idle animations
- Sleeping animation
- Click interactions
- Drag and move the cat
- Random behaviours
- Direction changes
- Multiple animation states

## 🛠 Tech Stack

- Electron
- HTML
- CSS
- JavaScript

## 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/YOUR_USERNAME/desktop-cat.git
```
Go to the project directory

```bash
cd desktop-cat
```
Install dependencies

```bash
npm install
```

Start the application

```bash
npm start
```
